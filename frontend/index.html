Cloud Architecture

Simple, secure architecture with a production-first mindset.

This portfolio is designed to communicate how the frontend fits into an AWS-native delivery model, which is useful for cloud, infrastructure, and platform engineering interviews.

  1. User accesses the site through Route 53 and CloudFront.
  2. CloudFront serves the static frontend from an S3 origin.
  3. A future visitor counter can call API Gateway and Lambda.
  4. Lambda stores counts in DynamoDB with IAM-based access control.
  5. CloudWatch is used for logs, metrics, and operational visibility.
